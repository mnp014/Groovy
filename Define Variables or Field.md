# Variables_Or_Fields.groovy
### Code:
```
    class GroovyABC{
        static void main(String[] args){
              def <variable1> = "<String Value>";     // variable are dynamically-typed, Allows the same to be assigned with different data-type.
              <variable1> = <int Value>
        }
    }
```
### Run:
```
groovy Variables_Or_Fields.groovy
```
___

### Example:
```
    class GroovyABC{
        static void main(String[] args){
              def abc = "xyz";                       // variable are dynamically-typed, Allows the same to be assigned with int value.
              abc = 25;
        }
    }
```
